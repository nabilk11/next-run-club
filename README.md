# Next Run Club aka "Runjida" Run App

Next Run Club is an social platform which allows runners to track, record and share their daily runs, statistics, and progress.

## Screens


/login

/dashboard

/new

/r
  /[username]


- User Dashboard
- Public Profile


## Data model

### Run 
- distance
- duration
- timestamp

### Runner
- email
- password
- username
- runs
- profile_pic_url
- friends
